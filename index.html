Sizes of C data types in IA32 IA32 does not provide hardware support for 64-bit integer arithmetic. Compiling code with long long data requires generating sequences of operations to perform the arithmetic in 32-bit chunks.

Registers

Registers are the fastest kind of memory available in the machine. x86-64 has 14 general purpose registers and several special-purpose registers.
